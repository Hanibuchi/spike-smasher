# Spike Smasher (スパイク・スマッシャー)

![サムネ](サムネ.gif)

**[🎮 今すぐブラウザでプレイ (unityroom)](https://unityroom.com/games/spike_smasher)**

## 概要
スパイクボールと繋がっている鎖の端をドラッグで操作し、制限時間内に街を破壊し尽くす爽快3Dアクションゲームです。
WebGL向けに制作されており、ブラウザですぐに遊ぶことができます。

## ゲームシステム
- **破壊と成長**: 街にあるものを破壊するとスコアが上がり、スコアに応じてスパイクボールが徐々に巨大化していきます。
- **耐久度と破壊条件**:
  - 各オブジェクトには「耐久力」が設定されており、スパイクボールのサイズが耐久力以上になると破壊可能です。
  - 最初は小さな物（コーンやベンチなど）しか壊せませんが、巨大化するにつれて最終的にはビルなどの巨大な建物も粉砕できるようになります。
  - ※ 破壊するには、スパイクボールが一定以上のスピードで激突する必要があります。
- **爽快な演出**: オブジェクトを破壊すると、その場にエフェクトと獲得スコアがポップアップ表示されます。

## 操作方法と物理挙動
- **ドラッグ操作**: マウス（または指）のドラッグ操作で鎖の端を動かしてスパイクボールを振り回します。
- **操作UI**: ドラッグしていない時は、直感的に操作が分かるよう、鎖の端付近（ワールド座標の中心）にスワイプ・ドラッグを促すUIが表示されます。
- **物理演算**: ドラッグ中は、鎖の端がY軸一定の状態で指の指す地点に移動します。スパイクボールと鎖は完全な物理演算で制御されており、`Rigidbody.MovePosition`によって自然な振り回し挙動を実現しています。

## カメラワーク (Cinemachine)
- 俯瞰視点・斜め上からの見下ろしカメラを採用しています。
- スパイクボールが巨大化するにつれて、カメラの映す範囲（FOVや距離）が自動的に広がり、ダイナミックなスケール感を楽しめます。

## ゲームフロー (1シーン構成)
1. **タイトル画面**: 「スタート」ボタンを押すとゲーム開始。タイマーのカウントダウンが始まり、操作が可能になります。
2. **メインゲーム**: 制限時間内にひたすら街を破壊してスコアを稼ぎます。時間が経過すると自動的に終了します。
3. **リザルト画面**: 獲得スコアが表示されます。「リスタート」ボタンで再挑戦、「ツイート」ボタンで結果をSNSでシェアできます。
